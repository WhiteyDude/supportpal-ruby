# supportpal-ruby

This gem provides direct calls to the SupportPal REST APIs

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'supportpal-ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install supportpal-ruby
